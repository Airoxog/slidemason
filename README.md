## What is slidemason?

slidemason is an open-source application that helps you create presentations quickly. It uses AI to turn your documents into clean, polished slides. You do not need any accounts or API keys. Everything happens locally on your Windows computer. This means your content stays private and safe.

The app works with popular AI tools like Cursor, Windsurf, Claude Code, or any AI coding agent you have. It supports common formats like PDF and PowerPoint. You do not need technical skills to get started.

## 📂 How to Use slidemason

1. **Import Your Document**  
   Click the “Import” button to select a document you want to use. slidemason supports PDF files and simple text documents.  

2. **Choose AI Agent**  
   Select Cursor, Claude Code, Windsurf, or any other AI agent you want to generate slides with. The app runs all AI processes locally; there is no need to enter API keys or sign in.  

3. **Generate Slides**  
   Press “Generate” to let the AI build your slide deck. Wait a few moments — the length and complexity of your document affect the time.  

4. **Review Slides**  
   Go through the slides in the preview section. You can make edits by clicking on any slide to adjust text or order.  

5. **Export Your Presentation**  
   When finished, click “Export” to save your presentation. Choose PowerPoint (`.pptx`) or PDF format.

## 🛠 Troubleshooting Tips

- **App won’t start:** Restart your computer and try again. Check if Windows Defender or antivirus blocked the app.  
- **Import errors:** Make sure your files are not damaged and are standard PDF or text formats.  
- **Slow slide generation:** Larger documents take longer. Close other apps to free up system memory.  
- **Export issues:** Check that you have write permission to the folder where you save slides.
